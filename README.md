# 🔐 Sistema de Autenticación de Usuarios (Login/Registro)

¡Bienvenido a mi proyecto de portafolio! Es una interfaz de inicio de sesión y registro de usuarios moderna, limpia y completamente responsiva. El proyecto cuenta con una arquitectura de servidor estructurada en Node.js, adaptada con simulaciones de cliente para su demostración estática en la web.

## 🛠️ Tecnologías y Herramientas Utilizadas
* **Node.js:** Configuración del entorno del lado del servidor y gestión de dependencias locales.
* **HTML5 & CSS3:** Estructura semántica del formulario y diseño visual responsivo (adaptable a móviles y computadoras).
* **JavaScript (Vanilla):** Manejo del DOM, validación de campos e intercepción de eventos para la simulación del flujo de autenticación en entornos estáticos.
